# 📽️ Teleprompter

一款專為手機設計的瀏覽器提詞器，邊看提示詞邊錄影，無需安裝任何 App。

---

## ✨ 主要功能

| 功能 | 說明 |
|------|------|
| 📜 提示詞滾動 | 依照 WPM 語速自動捲動，滾動速度實時可調 |
| 🎥 同步錄影 | 錄製畫面同步進行，錄完即可下載 MP4 |
| ⏱️ 預計時長 | 依字數與語速即時計算說完需要多少時間 |
| 🔁 重新錄製 | 錄影完成後可一鍵清除重錄，系統會提示確認 |
| 🪞 鏡像模式 | 支援鏡像翻轉，適合對著螢幕自拍 |
| 📷 前後鏡頭切換 | 一鍵切換前置／後置鏡頭 |
| 🔄 畫面旋轉 | 支援左轉 90°、右轉 90° 及還原直立 |
| 🔠 字體大小 | 設定中可調整字體大小（16–100px） |
| 🎨 文字顏色 | 自由選擇提示詞文字顏色 |
| ⏳ 倒數計時 | 錄影前可設定 0–10 秒倒數 |
| 💾 設定自動儲存 | 所有設定（含文字內容）存入手機瀏覽器，重開頁面自動還原 |
| 🌐 多語言 | 支援繁體中文、簡體中文、英文介面 |
| 🖱️ 浮窗可拖移／縮放 | 提示詞視窗可自由拖動位置與調整大小 |

---

## 📱 使用方式

1. 用手機 Safari 或 Chrome 開啟頁面（需 HTTPS）
2. 允許**相機**與**麥克風**授權
3. 點右上角 ⚙️ 開啟設定，輸入提示詞、調整語速與字體
4. 關閉設定，按底部紅色錄影鈕開始倒數錄製
5. 錄影結束後，設定抽屜會自動打開，點「下載影片」儲存 MP4

---

## 🎛️ 底部工具列說明

```
[ − WPM + ]   ⬤   [ 翻轉 ] [ 鏡像 ]
```

- **WPM 調節**：即時調整語速，提示詞滾動速度跟著變化
- **錄影鈕**（紅色圓形）：開始錄影 → 按下停止 → 再按可重新錄製
- **翻轉**：切換前置／後置鏡頭
- **鏡像**：開啟／關閉水平鏡像（藍色 = 開啟）

---

## ⚙️ 設定抽屜選項

- **語言**：繁體中文 / 簡體中文 / English
- **提示詞**：輸入你要讀的文字
- **預計時長**：自動計算
- **語速 WPM**：30–400，預設 120
- **字體大小**：16–100px，預設 30px
- **文字顏色**：顏色選擇器
- **倒數計時**：0–10 秒
- **旋轉**：逆時針 / 直立 / 順時針
- **下載影片**：錄影完成後顯示，點擊下載 MP4

---

## 💡 提示詞浮窗操作

- **拖動**：按住左下角 ✥ 圖示拖移浮窗位置
- **縮放**：拖動右下角縮放角標調整大小
- **開設定**：點右上角 ⚙️

---

## 🔧 技術細節

- 純前端，無後端、無框架、無安裝
- 使用 `MediaRecorder API` 錄製影片
- 使用 `localStorage` 自動儲存所有用戶設定
- 支援 iOS Safari 16+、Android Chrome、桌面瀏覽器
- 需要 HTTPS 環境才能使用相機與麥克風

---

## ⚠️ 已知限制

- iOS Safari 隱身模式下 `localStorage` 不可用，設定無法儲存（屬瀏覽器限制）
- 下載的影片格式為 WebM 容器封裝，副檔名為 `.mp4`，部分播放器需轉檔
- 錄影畫面為實際鏡頭畫面，不包含螢幕上的提示詞文字覆蓋層
